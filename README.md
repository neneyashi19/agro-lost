# 🎮 Agro-Lost - Explorando Mídia Perdida

Bem-vindo ao **Agro-Lost**, um site dedicado a explorar e documentar os mistérios da mídia desaparecida e esquecida da internet.

## O que é Lost Media?

**Lost Media** refere-se a conteúdo digital que desapareceu, foi deletado ou se tornou praticamente impossível de encontrar. Isso inclui:

- 🎮 **Jogos Desaparecidos**: Títulos removidos de lojas digitais
- 📺 **Vídeos Esquecidos**: Conteúdo deletado de plataformas
- 🌐 **Websites Extintos**: Sites que desapareceram completamente
- 🎬 **Produções Canceladas**: Filmes e séries nunca lançadas
- 🎵 **Músicas Perdidas**: Demonstrações e raridades

## 🚀 Features

- 🎨 **Design Moderno com Tema Cyberpunk**: Interface imersiva com efeitos de glitch
- 📱 **Responsivo**: Funciona perfeitamente em celulares e desktops
